Електронний посібник

з предмета "Матеріалознавство швейного виробництва" для професії 7433 Кравець за темою "Виробництво тканин"

Загальна інформація про посібник.

  

Електронний посібник - це електронне видання, що частково або повністю замінює чи доповнює підручник. На відміну від підручника, у ньому містяться три обов'язкових частини - класичний предметний зміст, вправи для закріплення матеріалу та контролю знань. Означений електронний засіб передбачає наявність певної методики, що розробляється в рамках педагогічного підходу (проблемного, проектного, контекстного, евристичного та ін.).

Електронний навчальний посібник з предмета: "Матеріалознавство швейного виробництва" за темою: "Ткацькі переплетення" розроблений в межах Державного стандарту ДСПТО 7433.С.14.10-2016 для підготовки кваліфікованих робітників з професії «кравець» (кваліфікація: кравець 2-3 розрядів) в середовищі Windows за допомогою  візуального веб-редактора Turbosite.

Електронний навчальний посібник "Ткацькі переплетення" створено з метою надання допомоги  вчителям, учням професіонально - технічних закладів під час вивчення теми, що вивчаються з професії «кравець». Електронний навчальний посібник покликаний сприяти кращому засвоєнню учнями навчального матеріалу, який вивчається.

 

Використання електронного посібника.

1.  Використовувати електронний посібник можна за допомогою комп’ютера та сучасного мобільного телефону.

2.  Використовувати електронний посібник можна:

а)  під час занять теоретичного навчання;

б) для самостійного вивчення матеріалу;

в) для проведення консультацій;

г)  для самоконтролю засвоєних знань;

д) для випереджального навчання;

е)  для дистанційного навчання.

3. Під час занять теоретичного навчання електронний посібник можна використовувати:

а)  на етапі актуалізації опорних знань: як ілюстративний матеріал;

б) на етапі вивчення нового навчального матеріалу: як унаочнення;

в) для організації самостійної, парної чи групової роботи учнів;

г)  на етапі узагальнення та систематизації знань;

д) як один із елементів домашнього завдання.
